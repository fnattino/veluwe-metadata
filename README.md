# Veluwe Metadata Repository

Test metadata repository for the Veluwe Metadata Portal.

Metadata records are stored using the [Metadata Control File (MCF)](https://geopython.github.io/pygeometa/reference/mcf/) format. Files can be validated using the [pygeometa](https://geopython.github.io/pygeometa/) CLI tool:

```shell
pygeometa metadata validate /path/to/mcf.yml
```
